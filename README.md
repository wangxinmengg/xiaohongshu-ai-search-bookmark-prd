# 小红书学习经验内容 AI 辅助搜索与收藏体验优化

本项目是一个 AI 产品优化方案，聚焦小红书中学习、考试、求职、留学类经验内容的搜索与收藏场景。

项目目标不是重做小红书搜索或收藏系统，而是在现有搜索结果浏览和收藏加入专辑路径中，加入轻量、可控、可跳过的 AI 辅助能力，降低用户判断单篇笔记价值和收藏后分类的决策成本。

## 项目背景

在学习 / 求职类内容场景中，用户通常需要点进多篇笔记，才能判断内容是否真正有用。收藏后，用户还需要判断应该加入哪个专辑，或是否需要新建专辑，容易造成分类成本和后续查找困难。

调研结果显示：

- 94.12% 用户至少偶尔需要点进多篇笔记判断内容价值
- 94.12% 用户认为 AI 内容要点至少有一定帮助
- 88.24% 用户至少偶尔遇到收藏后不好找的问题
- 76.47% 用户愿意或可能使用 AI 专辑推荐功能

## 核心问题

1. 单篇笔记内容价值判断成本高  
2. 收藏后加入专辑的分类决策成本高  

## 核心方案

### 1. AI 内容要点提示

在搜索结果卡片中展示 2–4 个 AI 提炼的内容要点，帮助用户在点进笔记前快速判断内容是否匹配需求。
